大 chotto crazy 2020 公式作品集 「こすって何」 を 発売 します。

2020年11月1日(日) 音ビルにて、大 chotto crazy 2020 に集まった実現したいことより、浅野充利／YeXuQo／seaketa／Kenny Pain の実現したいことを実現するための環境を用意し一日限り の 参加型アトラクション こす を開催しました。

YeXuQo・seaketaによるライブパフォーマンスをBGMに、芝生でくつろいだり、植物を植え替えたり、石を磨いたり、光る透明粘土で遊んだり 。
そこに核となる出し物はなく、ただそこにいるみんなが平等に遊び作ることを楽しむための場所と時間を作ることを目指しました。

この作品集は、YeXuQo／seaketa／Kenny Pain／ペフによる「こす」の感想と写真に加えて、ご家庭で「こす」を再現するためのいくつかのおまけを加えた 大 chotto crazy 2020 公式作品集 です。
