同じ内容・デザイン・制作方法でも印刷するほうが価値があるみたいな固定概念というか、考え方が気に食わない。

印刷したかったら自分で印刷しようと思うし、ただ持っていたかったらPDFでもいいと思う。

といっても、まだその行動が自分の習慣にはなってないし、そんな文化が日本でも根付けば、デジタルとアナログの価値基準がもっとフラットになると思うから、まずはウェブサイトを出力することを自分の習慣にできるように印刷・製本できる場所でそんな時間を作りたい。

Printed Web
by Kazuma Sasajima

2013年に、ウェブサイトは永久的なものではないという観点から、アメリカのグラフィックデザイナー Paul Soulellis が提唱した ウェブサイトを印刷しその価値を保存する Printed Web っていう文化を知ったことがこのプログラムを発案したきっかけです。

2017年にはニューヨークの現代美術館 MoMA に書籍と同等の価値がある資料としてPrinted Webのライブラリーが並んだことを知って、アメリカではウェブサイトを印刷するって文化が根付こうとしているんだということに衝撃をうけました。


Paul Soulellisが提案したPrinted Webっていう文化・習慣を身につけるように、自分の好きなウェブサイトをPDFに出力する／印刷・製本する時間を作りたい。


まだインターネットが普及する前はレンタルした音楽をカセットテープやMDにダビングしたり、自分好みのテレビ番組を録画して保存したり、自分の好きなものを集めたものを自分で作ってたと思う。

便利になりすぎたせいか、もうあるからその必要がないって思うのかもしれない。自分でつくるって気持ちが薄れているのが楽しくない。

これまでペフの出版物をPDFやウェブサイトなどの形式で発表したりもしたけど、印刷物のほうが価値があると受け取られてしまう感じはまだ変えられないと思う。

デジタルとアナログの価値の差を埋めるには、自分で印刷して製本するって文化を紹介すること、印刷にはどれだけコストと労力がかかるのかを体現することのほうが近道なんじゃないかなって気がしてきた。
このプログラムを通して、印刷して保存する／デジタルで保存する、それぞれのメリット・デメリットを理解し、デジタルとアナログの価値基準の差をもう少し狭めることに試みたい。
